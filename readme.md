
## Pinout Connection

**Raspberry Pi Pico 2 W** → **DB9 Adapter**

| Pico Pin | Signal | DB9 Adapter |
|---------|--------|-------------|
| PIN 40  | +5V    | VCC         |
| PIN 38  | GND    | GND         |
| PIN 1 (GP0) | UART TX | TX  |
| PIN 2 (GP1) | UART RX | RX  |

**Note:** Depending on the adapter, TX/RX may need to be crossed (TX → RX, RX → TX). Verify if required.
